For, es un statement que se usa para trabajar con bucles. Lo que hace es evaluar una expresión condicional dada y si ésta es verdadera-true- ejecuta una función dada, si la condición es falsa-false- la función no se va a ejecutar. La sintaxis es: for () {bloque de código }.Dentro de los paréntensis hay siempre tres elementos: la declaración de una variable; la condición de frenado; y la forma de ir incrementando la variable. Forma parte del Código DRY. Se usa para no tener que escribir muchas veces lo mismo, para evitar tareas repetitivas.
Hay operadores lógicos que son AND && ; OR || Y NOT !. Se usan en la lógica booleana. El && se usa para que me devuelva true si todas las condiciones dadas son true, ej: si a y b son true me devuelve true. El || se usa para que me devuelva true si alguna de las condiciones dadas es true, y si las condiciones son falsas me devuelve false, ej: si a es true o b es false me devuelve true. Y el ! me devuelve lo contrario, le da vuelta el significado, ej: si tengo que a no es igual a true me devuelve false.
